# 数组
  一种线性表数据结构。它用一组连续的内存空间，来存储一组具有相同类型的数据

## 线性表
  线性表就是数据排成像一条线一样的结构
  * 队列（先进先出）
  * 栈（先进后出，后进先出）
    * 注：非线性表（连续的内存空间和相同类型的数据）
    * 二叉树
    * 堆
    * 图
## 连续的内存空间和相同类型的数据
 * 优势：随机访问
 * 劣势：数组操作很低效（删除，插入）
## 为什么数组的下标从0开始
* 节省了一次cpu运算
  ```
  a[k]_address = base_address + k * type_size
  a[k]_address = base_address + (k-1)*type_size

  ```
